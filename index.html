Management. Made. Modular.

Managely is a modular server management plugin designed and built for modern versions of Minecraft. With powerful server utilities, simple to use gui and cli configuration options, and a clean, modern aesthetic that helps you cut through the mess of server moderation, Managely might be the plugin you need.
